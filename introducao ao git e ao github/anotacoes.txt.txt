Comandos para navegar dentro do windows "cmd".

1- CD / ==> navegar entre as pastas do disco C:
2- DIR ==> listar as pastas do disco
3- MKDIR ==> criar diretorio
4- DEL ==> deletar arquivo
5- RMDIR ==> remover todos os arquivos de dentro do repositorio
6- CD .. ==> voltar uma pasta anterior
7- ==> cls ou clear ==> comando para limpar a tela "cmd ou git bash
8- TAB ==> serve para autocompletar
9-arrow to up ==> serve para voltar ao comando anterior digitado no terminal
10- (-A)==> serve para mostrar pagina oculta.
